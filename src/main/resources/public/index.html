Im diesem Projekt wurde eine abgespekte Version von Socrative umgesetzt, also ein "Socrative Light". Das Socartive Light enthält folgende Funktionen:

  • Es kann entschieden werden, ob es sich um einen Lehrer oder Schüler handelt.
  • Der Lehrer kann Fragen erstellen Es gibt folgende Fragetypen: Schriftliche Kurzantworten und True/False(Is text question). Bei True/False-Fragen kann true oder false ausgewählt werden. Bei schriftlichen Kurzantworten kann der Schüler eine Antwort in ein Textfeld schreiben.
  • Ein Schüler sieht die vom Lehrer erstellten Fragen. Er kann die Fragen durchgehen und beantworten, am Ende de Quizzes werden die Fragen gesendet.
  • Der Lehrer kann ein Report ausgeben, in dem die Antworten aller Schüler ersichtlich sind.
